Sensore UNI interno di temperatura ed umidità

Sensore di temperatura e umidità per le reti LoRa WAN. È un sensore di classe A adatto per le reti con classe di protezione IP20.


  • Misura Temperatura ed Umidità
  • Richiede un alimentazione a + 5V con USB
  • Monitor
  • Contatore S0 Integrato
  • Risoluzione dell'umidità +/- 2% RH
  • Upgrade Firmware
  • Temperature a cui lavora da -5 ° C a + 65 ° C
  • Grado di Protezione IP20
