Enectiva REST API

Las librerías de Enectiva REST API permiten a todo el mundo usar la funcionalidad del sistema Enectiva sin tener que instalar Hardware para recopilar los datos. Gracias a la API, Enectiva puede estar integrado con sus sistemas empresariales, sistemas de autentificación, sitios web, o sistemas de gestión de edificaciones (BMS). Actualmente, la API esta disponible para el envio de lecturas de energía y series de tiempo personalizadas (por ejemplo, datos de estaciones meteorológicas) como está descrito más abajo.

Conecta Enectiva fácilmente con tu sistemas mediante una API

Casos de utilización

…con BMS

Los sistemas de gestión de edificaciones pueden transmitir datos desde los contadores directamente a los servidores de Enectiva. De esta manera, Enectiva puede integrarse fácilmente con el Hardware de otros fabricantes de sistemas de medición.

…con sistemas de autentificación

Los costes de facturación de energías pueden ser enviados directamente a Enectiva que posteriormente los utiliza para estadísticas de gestión energética. Nunca más necesitarás introducir esta información manualmente, ya que el proceso puede estar completamente automatizado.

…con sistemas ERP

Al igual que con el software de autentificación, Enectiva también puede estar integrado con otros sistemas empresariales. Enectiva es capaz de aprovechar los datos sobre la ocupación, producción, ventas, y otros parámetros en el contexto de la administración energética y mejorar el balance energético.

…con tu propia estación meteorológica

Sí quieres que los datos meteorológicos coincidan exactamente con los de su propiedad, su estación meteorológica puede estar conectada directamente a los servidores de Enectiva — puede enviarse datos de temperaturas que posteriormente se vinculan con su cuenta y las estadísticas de consumo de gas / calefacción.

Cómo funciona

Usar la API es muy simple: las lecturas son enviadas mediante peticiones HTTP, en otras palabras usando la tecnología, que sostiene la web y la cual puede ser usada fácilmente en cualquier entorno con acceso a Internet. La API verifica cada lectura recibida y la envía a Enectiva, donde las lecturas se reflejarán en estadísticas. Toda la documentación técnica contiene la información para más de una docena de lenguajes de programación; por lo tanto, la conexión del sistema existente con Enectiva es cuestión de solo unos minutos.

Para enviar lecturas con regularidad, es suficiente con conectar el código proporcionado a la fuente de los datos y enviar lecturas periódicamente.
