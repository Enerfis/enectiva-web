Enectiva REST API

Enectiva REST API Librarian позволяет каждому использовать функциональные возможности системы Enectiva без необходимости устанавливать соответствующее оборудование для автоматического сбора данных со счетчиков. Благодаря API можно интегрировать Enectiva с бизнес-системами, системами учета, сети или системами измерения и регулирования. В настоящее время полностью доступен API для отправки показаний энергии и собственных данных (например данных от метеорологических станций), как описано ниже.

Легко соедините Enectiva с вашей системой через API

Примеры использования

…с системами измерения и регулирования

Системы измерения и регулирования могут посылать показания датчиков непосредственно на Enectiva серверы. Таким образом, можно Enectiva также интегрировать с другими производителями аппаратных объектов, обеспечивающих показания счетчиков.

…с бухгалтерскими системами

Стоимость счета за коммунальные услуги могут быть направлены непосредственно на Enectiva и впоследствии использоваться для статистик, связанных с управлением энергией. Вам больше не нужно вводить эти данные вручную, этот процесс может быть полностью автоматизирован.

…с корпоративными системами ERP

Enectiva может интегрироваться как и с бухгалтерским программным обеспечением, так и с другими корпоративными системами. Enectiva умеет в рамках процесса управления энергетическими данными обрабатывать данные об объемах производства, занятости, маркетинга и другие параметры, которые имеют важное значение для любого энергетического бенчмаркинга.

            

…с собственной метеостанцией

Если вы хотите, чтобы ваши метеоданные точно отвечали вашему имуществу, ваша метеостанция может подключаться непосредственно к серверу Enectiva — отправлять данные о температуре, которые впоследствии связываются с Вашим счетом и статистиками потребления газа или тепла.

Как это работает

Использовать API очень просто: показания посылаются в HTTP-запросах, т.е. с использованием устройства, на котором есть веб-браузер и которое может подключиться к интернету. API проверяет каждое принятое показание и направляет его серверам Enectiva, впоследствие оно записывается в статистики. Полная техническая документация (EN) содержит код готовый к использованию более чем на дюжине различных языках программирования, таким образом, связывая Enectiva c существующей системой в течение нескольких минут.

Для регулярной отправки показаний необходимо предоставленный код подключить к источнику данных и передавать показания периодически.
