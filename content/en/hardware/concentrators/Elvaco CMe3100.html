Elvaco CMe3100

The CMe3100 is an M-Bus Metering Gateway for Fixed Network with logging functionality. It works with all ABB DIN-mounted electricity meters and any M-Bus meter that follows its standard protocol.



  • Works with any software that supports M-Bus.
  • Can send meter values via FTP, HTTP and email.
  • Add expansion modules as they are needed.
  • Configuration via web interface

Connections
  • x1 M-Bus master port (EN 13757); x2 M-Bus slave ports (EN 13757)
  • USB master port type A; USB slave port type miniB
  • RJ45(Ethernet)

Technical data
  • Dimensions (wxhxd) 70x90x64 (4DIN modules)
  • Weight: 190 g
  • Mounting on DIN rail 35mm
  • Power consumption: min <5W max <15W

Operating values:
  • Temperature: -25ºC to +55ºC
  • Humidity: 5% to 90%, non-condensing
