Elvaco CMe2100 (gen.3)

M-Bus metering Gateway for Mobile Network. It works with all ABB DIN-mounted electricity meters and any M-Bus meter that follows its standard protocol.



  • Works with any software that supports M-Bus.
  • Can send meter values via FTP, HTTP and email.
  • Add expansion modules as they are needed.

Connections
  • x1 M-Bus master port
  • USB master port type A; USB slave port type miniB
  • Network: Mobile (Radio)

Technical data
  • Dimensions (wxhxd): 35x100x65 mm
  • Weight: 120 g
  • Mounting on DIN rail 35mm
  • Antenna: SMA female
  • SIM-card: push-push type
  • Power consumpiton (max) <6W

Operating values:
  • Temperature: -20ºC to +55ºC
  • Humidity (max): 80% RH temperatures up to 31ºC decreasing linearly to 50% RH at 55ºC
