Indoor UNI temperature and humidity sensor

Temperature and Humidity sensor for LoRa WAN networks. It’s a class A sensor suitable for network with protection class IP20.


  • Temperature and humidity measurement
  • Watching + 5V presence on USB
  • Monitor contact status
  • Integrated S0 pulse counter
  • Moisture Sensor Resolution +/- 2% RH
  • Upgrade Firmware
  • Operating temperature -5 ° C to + 65 ° C
  • Degree of protection IP20
