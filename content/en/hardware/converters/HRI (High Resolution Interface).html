HRI (High Resolution Interface) - HRI-A and HRI-B Universal Converter

HRI (High Resolution Interface) is a universal converter that is compatible with a wide range of Sensus gauges. HRI is available in two versions. HRI-A is a high-resolution pulse unit. HRI-B is a data interface that supports M-Bus systems. The HRI design allows the system to be installed in extreme conditions such as water flooding. Remote reading is possible via M-Bus, radio, telephone or GSM.


HRI - B: Technical data
  • Cable length 1.5 m
  • Operating temperature range -10 째 C to + 65 째 C
  • Battery life for 12 years

HRI - B: Pulse Output
  • 2 pulse outputs according to ISO / TC30 / SC7 / WG8
  • Switching voltage max. 24 V
  • Switching current max. 20mA
  • Pulse length max. 124 ms
  • Memory for 1 million back pulses

HRI - B: Data interfaces
  • M-Bus and minibus (automatic)
  • Automatic baud rate detection (300/2400 Bd) and interface type
  • Data protocol according to IEC870 / EN 1434-3

HRI - B: Programming tools
  • MiniCom is software for HRI-B programming using PC with op. Windows
  • The software can be used with an M-Bus converter
