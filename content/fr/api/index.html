Enectiva REST API

Enectiva REST API Librarian permet à tous les utilisateurs de profiter de la fonctionnalité du système Enectiva sans qu'ils soient obligés d'installer un dispositif hardware approprié pour le rassemblement des données issues des compteurs. Grâce à API, il est possible de relier Enectiva avec vos systèmes d'entreprise, y compris avec le système comptable, le site, ou avec le système de mesure et régulation. Actuellement, nous offrons à votre pleine disposition API, qui sert à l'envoi des relevés énergétiques, comme il est décrit ci-dessous.

Relier simplement Enectiva avec vos système à l'aide d'API

Exemples de l'emploi

…avec les sytèmes de mesure et régulation

Il est possible que les sytèmes de mesure et régulation envoient les relevés des compteurs directement sur nos serveurs Enectiva. Il est ainsi possible de relier Enectiva avec d'autres dispositifs hardware des producteurs différents.

…avec le système comptable

Il est possible d'envoyer les frais issus des factures directement à Enectiva, qui s'en sert successivement pour l'élaboration des statistiques concernant le management énergétique. Vous ne serez plus obligé d'entrer les données manuellement, le processus est entièrement automatisé.

…avec les systèmes ERP d'entreprise

De la même manière qu'avec des logiciels comptables, il est possible de relier Enectiva avec d'autres systèmes d'entreprise. Enectiva sera ainsi capable, en cadre du management énergétique, élaborer les données sur la production, le niveau d'occupation, la vente et sur d'autres parametres importants pour le benchmarking énergétique.

…avec votre station météo

Si vous désirez que vos données météo correspondent exactement à votre bien immobilier, il est possible de relier votre station météo directement avec le serveur Enectiva — envoyer les données sur la température et avoir celles-ci successivement eliées avec votre compte et statistiques de la consommation du gaz et du chaleur.

Comment ça fonctionne

L'utilisation d'API est très simple: les relevés sont envoyés en protocole HTTP, c'est à dire, à l'aide de la technologie, qui est à la base des sites et qui est facilement applicable dans n'importe quel environnement muni de l'accès sur Internet. API contrôle chaque relevé reçu, pour le passer ensuite à Enectiva, ou il fera partie des statistiques. Documentation technique complète (EN) comprend un code préparé pour l'application dans une douzaine de langages de programmation, cela veut dire que l'interconnexion du système existant avec Enectiva ne prend que quelques minutes.

Pour un envoi régulier des relevés, il suffit de connecter le code préparé à la source des données et envoyer les relevés périodiquement.
