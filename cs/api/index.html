Enectiva REST API

Enectiva poskytuje REST API Librarian, která umožňuje automatizovat zadávání odečtů energií do Enectivy z jiných systémů.

Snadná Enectivy integrace s Vašimi systémy

Kdy je potřeba

  • Již sbíráte energetická data pomocí jiného systemu a chcete jej integrovat s Enectivou
  • Máte měřící hardware ve své kontrole a chcete využít Enectivu

Enectiva je nástroj pro energetický management a odečty energií jsou pro ni klíčové. Nabízíme instalaci a konfiguraci měřícího vybavení, o které se plně staráme, ale pokud již energie měříte a data máte k dispozici, stačí je posílat do Enectivy pomocí této API.

Jak to funguje

Použití API je velmi jednoduché: odečty se posílají v HTTP požadavcích, tj. pomocí technologie, na které stojí web a kterou lze snadno použít v jakémkoliv prostředí s přístupem k internetu. API zkontorluje každý přijatý odečet a předá ho Enectivě, kde se promítne do statistik. Kompletní technická dokumentace (EN) obsahuje připravený kód pro použití ve více než tuctu programovacích jazyků, tudíž propojení stávajícího systému s Enectivou je otázkou několika minut.

Pro pravidelné odesílání odečtů stačí poskytnutý kód napojit na zdroj dat a posílat odečty periodicky.
